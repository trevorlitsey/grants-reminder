# grants-reminder

Single-page app for organizing grant proposal deadlines. Sends automated emails as looming due-dates approach.

## Getting Started
```````
npm install
npm run start:dev
--> localhost:3000
```````

## Testing
````
npm run test
````

### Uses

* [Next.js](https://github.com/zeit/next.js/)
* [Firebase](https://firebase.google.com/)
